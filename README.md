# README.md

# Japanese Study Popup Extension

일본어 공부용 크롬 확장프로그램

일본어 단어를 드래그하면
마우스 근처에 한국어 번역 팝업을 표시합니다.

## Features

* 일본어 드래그 감지
* 한국어 번역 팝업
* 네이버 일본어사전 이동
* 복사 버튼
* 경량 UI

## Tech

* Chrome Extension Manifest V3
* Vanilla JavaScript
* Content Script 기반

## Roadmap

* 히라가나 표시
* 단어장 저장
* JLPT 레벨 표시
* 예문 표시
* AI 설명 기능

## Monetization

* 팝업 하단 고정 광고
* 쿠팡 파트너스 예정

## Install

1. chrome://extensions 이동
2. 개발자 모드 활성화
3. 압축해제된 확장 프로그램 로드
4. 프로젝트 폴더 선택
