# AGENTS

## Project Goal

일본어 단어를 드래그하면
한국어 번역 팝업을 띄우는
크롬 확장프로그램 개발.

## Current Features

* 일본어 드래그 감지
* 한국어 번역 표시
* 네이버 사전 버튼
* 복사 기능

## Development Rules

* Manifest V3 유지
* 가벼운 UI 유지
* 팝업 속도 우선
* 불필요한 라이브러리 사용 금지
* 한국어 사용자 UX 최우선

## UI Rules

* 팝업은 심플하게
* 광고는 하단에만 표시
* 학습 방해 요소 최소화

## Future Features

* 히라가나 표시
* 단어장 저장
* JLPT 레벨 표시
* 예문 표시
* 사용자 설정

## Git Workflow

작업 완료 시:

2. git add .
3. git commit 작성
