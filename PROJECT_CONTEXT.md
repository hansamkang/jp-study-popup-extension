# PROJECT CONTEXT

## Project Name

* 한글 이름: 일본어 드래그 번역
* 영문 이름: japanese-drag-translate

## Background

일본어 웹페이지를 읽는 사용자가 모르는 단어나 문장을 빠르게 확인할 수 있도록 만든 Chrome 확장 프로그램입니다.

사용자는 일본어 텍스트를 드래그하기만 하면 별도 페이지 이동 없이 마우스 근처에서 한국어 번역을 확인할 수 있습니다.

## Main UX

사용 흐름:

1. 웹페이지에서 일본어 텍스트를 드래그합니다.
2. 확장 프로그램이 일본어인지 확인합니다.
3. 마우스 근처에 작은 번역 팝업을 표시합니다.
4. 한국어 번역, 네이버 사전 링크, 복사 버튼을 제공합니다.
5. 팝업 하단에는 광고 영역을 표시합니다.

## Target Users

* 일본어를 공부하는 한국어 사용자
* 일본어 웹사이트를 자주 읽는 사용자
* JLPT 준비생
* 일본 취업이나 유학을 준비하는 사용자

## Core Value

* 빠른 일본어 의미 확인
* 한국어 중심 학습 경험
* 가벼운 사용성
* 웹페이지 읽기 흐름 방해 최소화

## Current Stack

* JavaScript
* Chrome Extension Manifest V3
* Content Script
* Background Service Worker

## Monetization

* 팝업 하단 광고 영역
* 추후 쿠팡파트너스 링크 적용 가능
* 광고 실패 시 번역 기능에는 영향이 없어야 함

## Notes

* UI는 최대한 단순하게 유지합니다.
* 번역 기능을 항상 최우선으로 유지합니다.
* 광고와 번역 기능은 서로 실패를 전파하지 않도록 분리합니다.
