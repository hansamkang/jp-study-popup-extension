# PROJECT CONTEXT

## Background

네이버 웨일의 일본어 사전 기능과 비슷한 경험을
크롬에서도 제공하기 위해 시작한 프로젝트.

한국어 기반 일본어 학습자용 확장프로그램.

## Main UX

사용자 흐름:

일본어 단어 드래그
↓
잠시 대기
↓
마우스 근처 팝업 표시
↓
한국어 번역 확인

## Target Users

* 한국인 일본어 학습자
* 일본 웹사이트 자주 보는 사용자
* JLPT 준비생
* 일본 취업 준비생

## Core Value

* 빠른 단어 확인
* 한국어 특화
* 가벼운 사용감
* 공부 흐름 방해 최소화

## Current Stack

* JavaScript
* Chrome Extension Manifest V3

## Future Direction

무료:

* 기본 번역
* 사전 이동

프리미엄:

* 히라가나

* 복습 기능
* AI 설명

## Monetization

* 팝업 하단 고정 광고
* 쿠팡 파트너스
* 향후 구독 모델 가능성 검토
